# Equipo 3 - Aplicación de Chat con IA

## Descripción
Aplicación de escritorio en Python tipo chat que permite cargar archivos,
realizar consultas con IA y exportar conversaciones en JSON/XML.

## Tecnologías
- Python
- Tkinter / PyQt
- SQLite
- PyPDF2 / pdfplumber

## Estructura del proyecto
- `src/` - Código fuente
- `docs/` - Documentación
- `diagrams/` - Diagramas UML
- `metrics/` - Formatos PSP y bitácoras
- `tests/` - Pruebas
- `assets/` - Archivos de prueba
